# Projet Portfolio Ana-Maria (Web)

**Objectif :** créer un site personnel pour présenter mon profil, mes compétences et quelques projets.  
**Outils :** WebStorm, HTML5, CSS3, Git, GitHub, GitHub Desktop.  
**Publication :** GitHub Pages.

**URL :** https://vassagia.github.io/Portfolio_Voisard_Ana_Maria_WEB/

---

## Concept & public cible

- **Concept :** portfolio multi-pages (accueil, CV, projets, contact).
- **Public cible :** recruteurs / entreprises (profil IT support / informatique de gestion) + évaluation ESIG.

---

## Objectifs pédagogiques

- HTML sémantique et structuré
- CSS moderne (variables, Grid/Flex, responsive)
- Accessibilité (navigation clavier, focus visible, textes alternatifs)
- Lisibilité en mode clair / sombre (prefers-color-scheme)

---

## Pages du site (racine)

- `index.html` : accueil (Hero + À propos)
- `cv.html` : CV (profil, compétences, expériences, formations)
- `projets.html` : cartes projets + liens vers GitHub
- `contact.html` : formulaire (simulation locale)

---

## Points techniques réalisés (résumé)

- Sémantique HTML (`header`, `nav`, `main`, `section`, `footer`, `article`, `figure`)
- Responsive (mobile-first + breakpoints)
- Variables CSS (couleurs, polices, espacements)
- Accessibilité : `skip-link`, `:focus-visible`, `.sr-only`
- Ressources locales : polices `.woff2`, images, icônes SVG
