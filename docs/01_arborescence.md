# Arborescence du projet

## Racine

- `index.html`
- `cv.html`
- `projets.html`
- `contact.html`
- `README.md`
- `docs/` : documentation de suivi

## Ressources

- `src/assets/css/`
    - `reset.css` : normalisation de base
    - `variables.css` : variables + polices locales
    - `base.css` : typographie, boutons, liens, accessibilité de base
    - `layout.css` : topbar, hero, sections, footer, responsive global
    - `pages.css` : styles spécifiques (projets, contact, CV) + dark mode
- `src/assets/img/` : images (hero + projets)
- `src/assets/icons/` : SVG (LinkedIn, GitHub)
- `src/assets/fonts/` : Inter en `.woff2`

## Documentation

- `00_intro.md` : présentation du projet
- `01_arborescence.md` : structure du projet
- `02_commits.md` : historique & logique des commits
- `03_css.md` : organisation CSS + choix techniques
- `04_bugs_et_solutions.md` : problèmes rencontrés + corrections
- `05_checklist_projet.md` : validations/tests (W3C, perf, a11y)
