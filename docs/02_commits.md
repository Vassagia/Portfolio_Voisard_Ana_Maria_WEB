# Historique Git / Commits

## Convention utilisée
- `feat:` nouvelle fonctionnalité
- `fix:` correction
- `style:` ajustements visuels / CSS
- `a11y:` accessibilité
- `docs:` documentation
- `chore:` maintenance

## Évolutions majeures
- Ajout des pages (accueil, CV, projets, contact)
- Mise en place du socle CSS (reset → variables → base → layout → pages)
- Accessibilité : skip-link, focus visible, sr-only
- Dark mode : lisibilité titres/textes/composants
- Uniformisation des cartes projets (même hauteur + images homogènes)
- Validation W3C HTML : suppression des patterns non conformes (slash inutiles sur void elements, rôles ARIA redondants)
- Ajout des liens projets vers GitHub (TimeHive / Mini-Tricorder)
